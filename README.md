# TaskMaster

TaskMaster é um gerenciador de tarefas moderno e intuitivo, construído com React e TypeScript. Projetado para ajudar você a organizar suas tarefas diárias de forma eficiente.

## 👨‍💻 Desenvolvedor

Desenvolvido por Tiago Cruz ([@tiagocruz.ia](https://www.instagram.com/tiagocruz.ia))

## ✨ Funcionalidades

- ✅ Criação, edição e exclusão de tarefas
- 🔍 Pesquisa de tarefas
- 🏷️ Sistema de prioridades (alta, média, baixa)
- 🎯 Filtros por status (todas, ativas, concluídas)
- 📋 Ordenação por data, alfabética ou prioridade
- 🌓 Tema claro/escuro
- 💾 Armazenamento local das tarefas
- ⌛ Registro de tempo desde a criação da tarefa

## 🚀 Tecnologias

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Shadcn/ui](https://ui.shadcn.com/)
- [React Router](https://reactrouter.com/)
- [React Query](https://tanstack.com/query/latest)

## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/taskmaster.git

# Entre no diretório
cd taskmaster

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

## 🔧 Scripts Disponíveis

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria a versão de produção
- `npm run build:dev` - Cria a versão de desenvolvimento
- `npm run preview` - Visualiza a versão de produção localmente
- `npm run lint` - Executa o linter para verificar o código
